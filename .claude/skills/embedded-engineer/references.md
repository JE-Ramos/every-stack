## RTFM — Essential Reading

### Books
- **Making Embedded Systems** by Elecia White — The best single-volume introduction to embedded engineering; covers hardware interfacing, firmware architecture, and the embedded mindset
- **The Art of Electronics** by Horowitz & Hill — The hardware fundamentals every firmware engineer needs; essential for understanding the circuits you are programming
- **Computer Systems: A Programmer's Perspective** (CS:APP) by Bryant & O'Hallaron — Deep understanding of how software interfaces with hardware at every level
- **Real-Time Systems** by Jane Liu — Formal treatment of scheduling theory and real-time system design
- **Test Driven Development for Embedded C** by James Grenning — Practical TDD in constrained environments

### Documentation & Guides
- [The Embedded Rust Book](https://docs.rust-embedded.org/book/) — Getting started with Rust on bare-metal ARM
- [FreeRTOS documentation](https://www.freertos.org/Documentation/RTOS_book.html) — Comprehensive RTOS concepts and API reference
- [Zephyr Project documentation](https://docs.zephyrproject.org) — Modern RTOS with excellent driver model and device tree support
- [ARM Cortex-M Architecture Reference](https://developer.arm.com/documentation/) — The authoritative reference for ARM processor programming
- [STM32 Reference Manuals](https://www.st.com/en/microcontrollers-microprocessors/stm32-32-bit-arm-cortex-mcus.html) — Exemplary peripheral documentation

### Articles & Blogs
- [Embedded Artistry](https://embeddedartistry.com) — Software engineering practices applied to embedded systems
- [Interrupt by Memfault](https://interrupt.memfault.com) — Deep technical blog covering firmware best practices, debugging, and tooling
- [Elecia White's Embedded.fm podcast](https://embedded.fm) — Interviews and discussions covering the breadth of embedded engineering
- [Ferrous Systems blog](https://ferrous-systems.com/blog/) — Embedded Rust in production, safety-critical systems
- [Jacob Beningo's blog](https://www.beningo.com/blog/) — Embedded software architecture and best practices

### Open Source Projects to Study
- [Zephyr RTOS](https://github.com/zephyrproject-rtos/zephyr) — Modern RTOS with excellent architecture; study the driver model and device tree integration
- [Embassy](https://github.com/embassy-rs/embassy) — Async Rust framework for embedded; the future of embedded Rust
- [Renode](https://github.com/renode/renode) — Hardware emulation framework for testing without physical devices

## References

- [IEC 62304](https://www.iso.org/standard/71604.html) — Medical device software lifecycle standard
