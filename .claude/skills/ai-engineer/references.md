## RTFM -- Essential Reading

### Documentation (Start Here)
- [Anthropic Documentation](https://docs.anthropic.com/) -- Claude API reference, prompt engineering guide, tool use, and best practices. The "Prompt Engineering" section is exceptional
- [OpenAI Cookbook](https://cookbook.openai.com/) -- Practical examples and best practices for building with OpenAI models. The RAG and evaluation examples are particularly valuable
- [LangChain Documentation](https://python.langchain.com/docs/) -- Comprehensive framework for LLM application development. Study the architecture patterns even if you don't use LangChain
- [LlamaIndex Documentation](https://docs.llamaindex.ai/) -- Data framework for LLM applications. Strong RAG patterns and retrieval strategies
- [Pinecone Learning Center](https://www.pinecone.io/learn/) -- Excellent tutorials on vector search, RAG, and embedding models

### Books & Courses
- **Building LLM Apps** (Valentina Alto) -- Practical guide to building LLM-powered applications with Python
- **AI Engineering** (Chip Huyen) -- Building applications with foundation models, covering the full production lifecycle
- [Full Stack LLM Bootcamp](https://fullstackdeeplearning.com/llm-bootcamp/) -- Practical course on building LLM-powered applications
- [DeepLearning.AI Short Courses](https://www.deeplearning.ai/short-courses/) -- Free courses on RAG, agents, evaluation, and AI engineering topics

### Articles & Resources
- [Anthropic Prompt Engineering Guide](https://docs.anthropic.com/en/docs/build-with-claude/prompt-engineering/overview) -- The best prompt engineering documentation available. Covers techniques from basic to advanced
- [Eugene Yan's Writing on Evaluation](https://eugeneyan.com/writing/evals/) -- Foundational article on evaluating LLM systems
- [Jason Wei on Chain-of-Thought](https://arxiv.org/abs/2201.11903) -- The original chain-of-thought prompting paper
- [LMSYS Chatbot Arena](https://chat.lmsys.org/) -- Live model comparison leaderboard — essential for understanding relative model capabilities
- [Simon Willison's Blog](https://simonwillison.net/) -- Prolific writing on LLM capabilities, limitations, and practical applications
- [Latent Space Podcast](https://www.latent.space/) -- AI engineering podcast covering the latest techniques and industry developments
- [AI Engineer Foundation](https://www.ai.engineer/) -- Community and resources for AI engineers

### Evaluation & Safety
- [Promptfoo Documentation](https://www.promptfoo.dev/docs/intro/) -- Open-source LLM evaluation framework. Red teaming, benchmarking, and CI/CD for prompts
- [RAGAS Documentation](https://docs.ragas.io/) -- Evaluation framework specifically for RAG pipelines
- [OWASP LLM Top 10](https://owasp.org/www-project-top-10-for-large-language-model-applications/) -- Security risks specific to LLM applications
- [Anthropic Safety Research](https://www.anthropic.com/research) -- Research on AI safety, alignment, and responsible deployment

### Open Source Projects to Study
- [LangChain](https://github.com/langchain-ai/langchain) -- The most widely adopted LLM application framework
- [LlamaIndex](https://github.com/run-llama/llama_index) -- Data framework for LLM applications with strong RAG support
- [Promptfoo](https://github.com/promptfoo/promptfoo) -- LLM evaluation and red-teaming framework
- [Langfuse](https://github.com/langfuse/langfuse) -- Open-source LLM observability and analytics
- [Guardrails AI](https://github.com/guardrails-ai/guardrails) -- Input/output validation framework for LLM applications
- [Chroma](https://github.com/chroma-core/chroma) -- Open-source embedding database for AI applications
