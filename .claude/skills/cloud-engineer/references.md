## RTFM -- Essential Reading

### Books
- **Cloud Native Patterns** (Cornelia Davis) -- Designing applications that leverage cloud characteristics. Essential for understanding cloud-native vs. cloud-hosted
- **AWS Well-Architected Framework** (AWS) -- The six pillars: operational excellence, security, reliability, performance efficiency, cost optimization, sustainability. Free whitepaper
- **Cloud Strategy: A Decision-based Approach to Successful Cloud Migration** (Gregor Hohpe) -- Strategic thinking about cloud adoption for architects and decision-makers
- **Networking and Kubernetes** (James Strong, Vallery Lancey) -- Deep dive into cloud networking for container orchestration
- **Cloud FinOps** (J.R. Storment, Mike Fuller) -- The definitive guide to cloud financial management, written by FinOps Foundation founders

### Documentation & Architecture Centers
- [AWS Architecture Center](https://aws.amazon.com/architecture/) -- Reference architectures, best practices, and pattern libraries from AWS
- [Azure Architecture Center](https://learn.microsoft.com/en-us/azure/architecture/) -- Design patterns, reference architectures, and best practices from Microsoft
- [Google Cloud Architecture Center](https://cloud.google.com/architecture) -- Architecture frameworks, design guides, and reference implementations from GCP
- [AWS Well-Architected Framework](https://docs.aws.amazon.com/wellarchitected/latest/framework/welcome.html) -- The six pillars with detailed guidance and review questions
- [FinOps Foundation Documentation](https://www.finops.org/framework/) -- The FinOps framework for cloud financial management

### Articles & Resources
- [The Amazon Builders' Library](https://aws.amazon.com/builders-library/) -- How Amazon builds and operates software. Deep technical articles on distributed systems, deployment, and operations
- [Last Week in AWS](https://www.lastweekinaws.com/) -- Corey Quinn's newsletter covering AWS news, costs, and honest cloud commentary
- [Cloud Security Alliance Guidance](https://cloudsecurityalliance.org/research/guidance) -- Comprehensive cloud security framework and best practices
- [InfoQ Cloud Articles](https://www.infoq.com/cloud-computing/) -- In-depth technical articles on cloud architecture and operations
- [CIS Benchmarks](https://www.cisecurity.org/cis-benchmarks) -- Security configuration benchmarks for AWS, Azure, and GCP

### Certifications (for structured learning, not checkbox collecting)
- AWS Solutions Architect Professional -- Deep AWS architecture knowledge
- Google Cloud Professional Cloud Architect -- GCP architecture and design patterns
- Azure Solutions Architect Expert -- Azure architecture and integration patterns
- FinOps Certified Practitioner -- Cloud financial management fundamentals

### Open Source Projects to Study
- [Prowler](https://github.com/prowler-cloud/prowler) -- Cloud security assessment tool for AWS, Azure, and GCP
- [Infracost](https://github.com/infracost/infracost) -- Cloud cost estimation for Terraform, integrated into CI/CD
- [CloudQuery](https://github.com/cloudquery/cloudquery) -- Cloud asset inventory and compliance framework
- [Steampipe](https://github.com/turbot/steampipe) -- Query cloud infrastructure using SQL
- [Checkov](https://github.com/bridgecrewio/checkov) -- Static analysis for IaC security and compliance
